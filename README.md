# daniel tts-server

SSML Web API that can be run on a Windows virtual server like EC2 and produces the [Daniel UK](https://www.youtube.com/watch?v=AW5EZbLzwnw) TTS voice.

## preqs

- Windows 64-bit
- Opened port 80
- Node.js (v16 works)

## install

1. Clone this repo
2. run "Daniel 22Khz MLG voice.exe" to install the voice
3. run `npm start`
